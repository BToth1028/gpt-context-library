## Topic Overview

The Project Context OS is a 4-portal engineering workspace designed to reduce AI context/token costs by 70-90%. It consists of:
- MkDocs (Documentation portal)
- Backstage (Service catalog)
- Sourcegraph (Code search)
- Structurizr (Architecture diagrams)

Plus supporting systems:
- Hot Context Builder (auto-generates .cursor/rules/context-hot.mdc)
- VECTOR_MGMT (chat history indexing)
- Automation scripts

## Context

This system was built to eliminate manual context explanation, reduce token costs ($7K-20K/year savings), and maintain institutional knowledge permanently.
